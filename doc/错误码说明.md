# Nuist CampusWall 错误码说明（V3.1）

## 1. 错误码表
| code | 常量 | 说明 | 常见触发场景 |
|---|---|---|---|
| 0 | - | 成功 | 请求正常完成 |
| 401 | USERNAME_EXISTS | 用户名已存在 | 注册用户名重复 |
| 402 | EMAIL_EXISTS | 邮箱已存在 | 注册/修改邮箱重复 |
| 403 | USERNAME_NOT_FOUND | 用户名不存在 | 登录账号不存在 |
| 404 | USER_DISABLED | 用户被禁用 | 账号被禁用后登录 |
| 405 | PASSWORD_INCORRECT | 密码错误 | 登录或改密校验失败 |
| 406 | UNAUTHORIZED | 未登录/缺 token | 访问受保护接口未携带 token |
| 407 | TOKEN_INVALID | token 无效或过期 | token 篡改、过期、格式非法 |
| 408 | USER_NOT_FOUND | token 对应用户不存在 | token 合法但用户不存在 |
| 409 | POST_NOT_FOUND | 帖子不存在 | 帖子 ID 无效 |
| 410 | COMMENT_NOT_FOUND | 评论不存在 | 评论 ID 无效 |
| 411 | LIKE_EXIST | 重复点赞 | 对同目标重复点赞 |
| 412 | NO_PERMISSION | 无权限 | 越权访问、普通用户访问管理员接口 |
| 413 | POST_STATUS_ERROR | 帖子状态错误 | 对不可操作状态帖子更新 |
| 414 | FILE_NOT_FOUND | 文件不存在 | 绑定/迁移目标文件不存在 |
| 415 | FILE_TYPE_ERROR | 文件类型错误 | 上传或绑定 fileType 不匹配 |
| 416 | FILE_SIZE_ERROR | 文件大小错误 | 超过业务限制（5MB） |
| 417 | FILE_STATUS_ERROR | 文件状态错误 | 重复绑定同一文件 |
| 418 | FILE_UPLOAD_ERROR | 文件上传失败 | 落盘或迁移失败 |
| 419 | FILE_DELETE_ERROR | 文件删除失败 | 清理物理文件失败 |
| 422 | PARAMETER_ERROR | 参数错误 | 参数校验失败、非法状态操作 |
| 500 | SERVER_ERROR | 服务器异常 | 未知异常兜底 |

## 2. 管理端常见码
1. 未登录：`406`
2. 非管理员：`412`
3. 资源不存在：`408/409/410`
4. 重复启用/禁用：`422`

## 3. 文件模块常见码
1. 非图片上传：`415`
2. 超过 5MB：`416`
3. 文件重复绑定：`417`
4. 文件迁移失败：`418`
5. 文件删除失败：`419`

## 4. 维护约定
1. `src/main/java/com/nuist_campuswall/common/ErrorCode.java` 是唯一真源。
2. 新增或调整错误码时必须同步：
   - `doc/API文档.md`
   - `doc/错误码说明.md`
   - `src/test/http/*.http` 的预期说明
