# Nuist CampusWall 错误码说明（V3.4）

## 1. 协议规则
1. 返回体：`{ code, message, data }`。
2. `code=0` 代表成功，非 0 代表业务失败。
3. HTTP 状态码表示失败类别。

## 2. 常见业务码
| code | 常量 | 含义 |
|---|---|---|
| 401 | USERNAME_EXISTS | 用户名已存在 |
| 402 | EMAIL_EXISTS | 邮箱已存在 |
| 405 | PASSWORD_INCORRECT | 密码错误 |
| 406 | UNAUTHORIZED | 未登录/缺token |
| 407 | TOKEN_INVALID | token无效 |
| 409 | POST_NOT_FOUND | 帖子不存在 |
| 410 | COMMENT_NOT_FOUND | 评论不存在 |
| 411 | LIKE_EXIST | 重复点赞 |
| 412 | NO_PERMISSION | 无权限 |
| 414 | FILE_NOT_FOUND | 文件不存在 |
| 415 | FILE_TYPE_ERROR | 文件类型错误 |
| 416 | FILE_SIZE_ERROR | 文件大小错误 |
| 417 | FILE_STATUS_ERROR | 文件状态错误 |
| 422 | PARAMETER_ERROR | 参数错误 |
| 500 | SERVER_ERROR | 服务器异常 |

## 3. HTTP 映射示例
1. `401`：`405/406/407`
2. `403`：`412`
3. `404`：`408/409/410/414`
4. `409`：`401/402/411/413/417`
5. `422`：`422`
6. `500`：`418/419/500`

## 4. 真源位置
1. 业务码：`src/main/java/com/nuist_campuswall/common/ErrorCode.java`
2. 映射器：`src/main/java/com/nuist_campuswall/common/ErrorCodeToHttpStatus.java`
3. 异常出口：`src/main/java/com/nuist_campuswall/common/GlobalExceptionHandler.java`
