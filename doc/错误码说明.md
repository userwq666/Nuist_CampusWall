# Nuist CampusWall 错误码说明（V2.3）

| code | 常量 | 说明 | 常见触发场景 |
|---|---|---|---|
| 0 | - | 成功 | 请求正常完成 |
| 401 | USERNAME_EXISTS | 用户名已存在 | 注册时用户名重复 |
| 402 | EMAIL_EXISTS | 邮箱已存在 | 注册时邮箱重复 |
| 403 | USERNAME_NOT_FOUND | 用户名不存在 | 登录账号不存在 |
| 404 | USER_DISABLED | 用户被禁用 | 账号状态不可用 |
| 405 | PASSWORD_INCORRECT | 密码错误 | 登录密码不匹配 |
| 406 | UNAUTHORIZED | 未登录/缺 token | 访问受保护接口未携带 token |
| 407 | TOKEN_INVALID | token 无效或过期 | token 篡改、过期 |
| 408 | USER_NOT_FOUND | token 对应用户不存在 | token 合法但用户已不存在 |
| 409 | POST_NOT_FOUND | 帖子不存在 | 帖子 id 无效或已删除不可用 |
| 410 | COMMENT_NOT_FOUND | 评论不存在 | 评论 id 无效或已删除不可用 |
| 411 | LIKE_EXIST | 重复点赞 | 同用户对同目标重复点赞 |
| 412 | NO_PERMISSION | 无权限 | 修改/删除他人资源 |
| 413 | POST_STATUS_ERROR | 帖子状态错误 | 对状态不允许的帖子操作 |
| 422 | PARAMETER_ERROR | 参数错误 | 参数校验失败 |
| 500 | SERVER_ERROR | 服务器异常 | 未知异常兜底 |

## 维护要求
1. 以 `ErrorCode.java` 为唯一真源。
2. 变更错误码时必须同步 `API文档.md` 与 `src/test/http/*.http` 预期说明。
