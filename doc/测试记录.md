# Nuist CampusWall 测试记录（V2.3）

## 1. 测试方式
1. 使用 IDEA HTTP Client。
2. 用例文件位于 `src/test/http/`。
3. 先执行 account 登录拿 token，再执行 post/comment/like。

## 2. 已覆盖场景
### 2.1 account
1. 注册成功。
2. 用户名重复。
3. 邮箱重复。
4. 登录成功。
5. 登录密码错误。
6. `/account/my` 带 token 成功。
7. `/account/my` 不带 token 失败。

### 2.2 post
1. 创建成功。
2. 未登录创建失败。
3. 参数非法失败。
4. 公开分页成功。
5. 我的分页成功。
6. 详情成功。
7. 修改成功。
8. 修改参数非法。
9. 修改不存在帖子。
10. 删除成功。
11. 删除不存在帖子。
12. 删除他人帖子（无权限）。

### 2.3 comment
1. 评论成功。
2. 未登录评论失败。
3. 参数非法失败。
4. 目标帖子不存在。
5. 评论分页成功。
6. 评论分页参数非法。
7. 我的评论分页成功。
8. 删除我的评论成功。
9. 删除不存在评论。
10. 删除他人评论（无权限）。

### 2.4 like
1. 点赞成功。
2. 重复点赞。
3. 取消点赞成功。
4. 重复取消点赞。
5. 未登录点赞失败。
6. 点赞不存在帖子。
7. 点赞不存在评论。
8. 取消点赞不存在评论。

## 3. 关键结论
1. 主链路可跑通：注册 -> 登录 -> 发帖 -> 评论 -> 点赞。
2. 公开数据受 `status=ENABLE` 过滤。
3. 软删除逻辑生效，权限校验返回稳定。
4. 业务错误码与接口文档一致。
