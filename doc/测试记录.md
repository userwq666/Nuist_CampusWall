# Nuist CampusWall 测试记录（V1.2）

## 1. 测试范围
本轮覆盖账户模块注册与登录接口：
1. `POST /api/account/register`
2. `POST /api/account/login`
3. `GET /api/account/me`

## 2. 测试环境
1. 本地开发环境（Windows）
2. Spring Boot 本地运行（默认 `8080`）
3. MySQL 本地实例
4. 测试工具：IntelliJ HTTP Client（`src/test/http/account.http`）

## 3. 用例记录

### 用例1：注册成功
请求：
```json
{
  "username": "test1001",
  "password": "123456",
  "nickname": "测试用户1001",
  "educationEmail": "test1001@nuist.edu.cn"
}
```

预期：
1. 返回 `code=0`
2. 返回成功消息
3. 数据库新增用户记录

结果：
1. 通过

## 4. 数据验证
1. 用户密码为 BCrypt 加密串，不是明文保存。
2. 用户名与邮箱重复提交会被拦截，不会写入重复数据。
3. 登录成功返回 token 与用户基础信息，不返回密码。
4. `/api/account/me` 依赖 token 鉴权，未携带或无效 token 会被拦截。

## 5. 结论
账户模块（注册/登录/JWT鉴权/me）闭环已可用，具备进入帖子模块开发条件。
