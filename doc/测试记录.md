# Nuist CampusWall 测试记录（V2.0）

## 1. 测试范围
1. 账户：注册、登录、当前用户
2. 帖子：创建、分页、详情
3. 评论：创建、分页
4. 鉴权：带 token / 缺 token / 无效 token
5. 参数校验：DTO 注解校验返回 `422`

## 2. 测试环境
1. 时间：2026-03-25
2. 后端：Spring Boot 本地启动，`http://localhost:8080`
3. 数据库：MySQL 8（`campuswall`）
4. 工具：IntelliJ HTTP Client
5. 用例文件：
   - `src/test/http/account.http`
   - `src/test/http/post.http`
   - `src/test/http/comment.http`

## 3. 用例与结果
### 3.1 账户模块
1. 注册成功：通过（`code=0`）
2. 用户名重复：通过（`code=401`）
3. 邮箱重复：通过（`code=402`）
4. 登录成功：通过（`code=0`，返回 `token + userInfo`）
5. 登录失败-密码错误：通过（`code=405`）
6. `/me` 带 token：通过（`code=0`）
7. `/me` 不带 token：通过（`code=406`）
8. `/me` 伪造 token：通过（`code=407`）

### 3.2 帖子模块
1. 创建帖子（带 token）：通过（`code=0`）
2. 创建帖子（不带 token）：通过（`code=406`）
3. 创建帖子（标题为空）：通过（`code=422`）
4. 创建帖子（标题超长）：通过（`code=422`）
5. 帖子分页：通过（`code=0`，返回 `PageResult<PostVO>`）
6. 分页参数非法：通过（`code=422`）
7. 帖子详情存在：通过（`code=0`）
8. 帖子详情不存在：通过（`code=409`）

### 3.3 评论模块
1. 创建评论（带 token）：通过（`code=0`）
2. 创建评论（不带 token）：通过（`code=406`）
3. 创建评论（参数非法）：通过（`code=422`）
4. 创建评论（目标帖子不存在）：通过（`code=409`）
5. 评论分页：通过（`code=0`，返回 `PageResult<CommentVO>`）
6. 评论分页参数非法：通过（`code=422`）

## 4. 结论
1. 当前“账户 + 鉴权 + 帖子 + 评论”主链路可用。
2. 业务错误码与文档一致。
3. 可进入下一阶段：点赞模块与管理模块开发。

## 5. 后续测试建议
1. 增加自动化测试（JUnit + SpringBootTest）。
2. 增加并发场景测试（点赞幂等、重复提交）。
3. 增加异常路径测试（数据库异常、token 过期边界）。
