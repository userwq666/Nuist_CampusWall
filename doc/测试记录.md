# Nuist CampusWall 测试记录（V2.1）

## 1. 测试范围
1. 账户：注册、登录、`/me`
2. 帖子：创建、分页、详情
3. 评论：创建、分页
4. 点赞：点赞、取消点赞

## 2. 测试环境
1. 时间：2026-03-26
2. 后端：`http://localhost:8080`
3. 数据库：MySQL 8
4. 工具：IntelliJ HTTP Client

## 3. 已通过用例
### 3.1 账户
1. 注册成功（`code=0`）
2. 用户名重复（`code=401`）
3. 邮箱重复（`code=402`）
4. 登录成功（`code=0`）
5. 密码错误（`code=405`）
6. `/me` 带 token（`code=0`）
7. `/me` 不带 token（`code=406`）

### 3.2 帖子
1. 创建成功（`code=0`）
2. 未登录创建（`code=406`）
3. 参数非法（`code=422`）
4. 分页成功（`code=0`）
5. 详情不存在（`code=409`）

### 3.3 评论
1. 创建成功（`code=0`）
2. 未登录创建（`code=406`）
3. 参数非法（`code=422`）
4. 目标帖子不存在（`code=409`）
5. 分页成功（`code=0`）

### 3.4 点赞（新增）
基于 `src/test/http/like.http`：
1. 点赞成功（`code=0`）
2. 重复点赞（`code=411`）
3. 取消点赞成功（`code=0`）
4. 重复取消点赞（`code=422`）
5. 未登录点赞（`code=406`）
6. 点赞目标不存在（`POST -> code=409`）

## 4. 结论
当前“账户 + JWT + 帖子 + 评论 + 点赞”链路可用，满足当前阶段课程演示要求。
