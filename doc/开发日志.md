# Nuist CampusWall 开发日志

## 2026-03-18（帖子模块起步）

### 1. 帖子创建链路完成
1. 新增 `POST /api/post/create` 接口。
2. 新增帖子模块基础层：
   - `dto/post/CreatePostDTO`
   - `mapper/post/PostMapper`
   - `service/post/PostService + PostServiceImpl`
   - `controller/post/PostController`
3. 发帖逻辑完成：
   - 从 `UserContext` 读取当前用户ID
   - 组装 `Post` 并写入数据库
   - 默认状态 `ENABLE`、点赞数 `0`

### 2. 参数校验接入
1. `CreatePostDTO` 增加校验注解：
   - 标题：非空，长度<=50
   - 正文：非空，长度<=10000
2. `PostController.create` 增加 `@Valid`。
3. `GlobalExceptionHandler` 增加参数校验异常处理，返回 `code=422`。

### 3. 测试完成
1. 带 token 发帖成功。
2. 不带 token 发帖拦截（`406`）。
3. 标题为空/超长触发参数校验失败（`422`）。

## 2026-03-18

### 1. JWT 与鉴权闭环完成
1. JWT 基础设施完成：
   - 新增 JWT 依赖（`jjwt-api/jjwt-impl/jjwt-jackson`）
   - `application.properties` 增加 `jwt.secret/jwt.expire-seconds/jwt.issuer`
   - 新增 `security/JwtUtil`（签发与解析）
2. 登录响应升级：
   - 新增 `LoginRespVO`（`token + userInfo`）
   - `POST /api/account/login` 返回 `Result<LoginRespVO>`
3. 鉴权拦截完成：
   - 新增 `security/JwtAuthInterceptor`
   - 新增 `security/UserContext`（ThreadLocal 保存 userId）
   - 新增 `config/WebMvcConfig` 注册拦截器并放行 `/api/account/register`、`/api/account/login`
4. 当前用户接口完成：
   - 新增 `GET /api/account/me`
   - 根据 token 中 userId 获取当前用户信息
5. 错误码常量化：
   - 新增 `common/ErrorCode`
   - 统一使用 `401~408、422、500`

### 2. 验证结果
1. 登录成功可返回 token 与 userInfo。
2. 带 token 访问 `/api/account/me` 成功。
3. 不带 token 访问 `/api/account/me` 返回业务错误码 `406`。
4. 伪造或过期 token 返回业务错误码 `407`。
