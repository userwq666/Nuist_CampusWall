# Nuist CampusWall 需求说明（V2.5）

## 1. 项目范围
1. 用户端：注册、登录、个人信息、发帖、评论、点赞、我的帖子/评论。
2. 管理端：权限校验、用户管理、帖子管理、评论管理。

## 2. 当前已实现需求
1. 注册、登录、JWT 鉴权、当前用户接口。
2. 帖子：公开分页、公告分页、我的分页、详情、修改、删除。
3. 评论：创建、分页、我的分页、删除。
4. 点赞：点赞、取消点赞。
5. 管理员：`/api/admin/ping`、`/api/admin/user/page`。

## 3. 核心业务规则
1. 公开帖子列表排除管理员公告帖子。
2. 公告列表只展示管理员帖子。
3. 管理员ID由配置项 `app.admin-user-id` 指定。
4. 公开列表只展示 `status=ENABLE`。
5. 删除采用软删除（`status=DISABLE`）。

## 4. 当前未实现需求
1. 管理员用户启用/禁用。
2. 管理员帖子删除/恢复。
3. 管理员评论删除/恢复。
4. 自动化测试体系。

## 5. 验收标准（当前阶段）
1. 用户主链路可完整演示。
2. 公告与普通帖子可分流展示。
3. 管理员基础鉴权链路可演示。
4. 文档与接口、用例一致。
