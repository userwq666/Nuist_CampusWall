# Nuist CampusWall 部署说明（开发环境）

## 1. 环境要求
1. JDK 21
2. Maven 3.9+
3. MySQL 8
4. IntelliJ IDEA（推荐）

## 2. 本地配置
编辑 `src/main/resources/application.properties`：
1. `spring.datasource.url`
2. `spring.datasource.username`
3. `spring.datasource.password`
4. `server.port`（默认 `8080`）

JWT 配置：
1. `jwt.secret`
2. `jwt.expire-seconds`
3. `jwt.issuer`

## 3. SQL 初始化
项目启动自动执行：
1. `classpath:sql/db_init.sql`
2. `classpath:sql/db_seed.sql`

说明：
1. `db_init.sql` 负责建表与约束
2. `db_seed.sql` 负责初始化管理员数据

## 4. 启动命令
在项目根目录执行：
```bash
mvn spring-boot:run
```

## 5. 快速验收
### 5.1 HTTP 环境变量
编辑 `src/test/http/http-client.env.json`：
```json
{
  "dev": {
    "host": "http://localhost:8080",
    "token": "<登录后token>"
  }
}
```

### 5.2 用例执行顺序
1. 执行 `account.http` 登录并拿到 token
2. 将 token 写回 `http-client.env.json` 的 `dev.token`
3. 执行 `post.http`
4. 执行 `comment.http`

## 6. 常见问题
### 6.1 端口占用
现象：`Port 8080 was already in use`
处理：修改 `server.port` 或关闭占用进程。

### 6.2 接口地址写成 `http:///api/...`
现象：`UnknownHostException: http:`
处理：检查 `host` 环境变量是否为空，并在 HTTP Client 中选择 `dev` 环境。

### 6.3 启动后没建表
处理：
1. 检查数据库连接信息
2. 检查 `spring.sql.init.*` 配置
3. 查看控制台 SQL 初始化日志

### 6.4 中文乱码
处理：
1. IDEA 文件编码统一 UTF-8
2. 终端切换 UTF-8（Windows 下执行 `chcp 65001`）

## 7. 交付建议
1. 提交前执行一次 `mvn -DskipTests compile`
2. 确保文档、HTTP 用例、代码三者一致
