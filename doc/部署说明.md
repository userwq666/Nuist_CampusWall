# Nuist CampusWall 部署说明（开发环境）

## 1. 环境准备
1. JDK 21
2. Maven 3.9+
3. MySQL 8.x
4. IDEA（UTF-8 编码）

## 2. 数据库准备
1. 创建数据库：`campuswall`
2. 启动时自动执行：
   - `src/main/resources/sql/db_init.sql`
   - `src/main/resources/sql/db_seed.sql`

## 3. 配置项
编辑 `src/main/resources/application.properties`：
1. `spring.datasource.url`
2. `spring.datasource.username`
3. `spring.datasource.password`
4. `jwt.secret`
5. `jwt.expire-seconds`
6. `jwt.issuer`

## 4. 启动命令
```bash
mvn spring-boot:run
```

## 5. 本地回归顺序
1. `src/test/http/account.http`
2. `src/test/http/post.http`
3. `src/test/http/comment.http`
4. `src/test/http/like.http`
5. `src/test/http/admin.http`

## 6. admin 测试补充
1. `http-client.env.json` 需要配置 `adminToken`。
2. 普通用户 token 对管理员接口应返回 `412`。

## 7. 常见问题
1. `http:///api/...`：检查 `http-client.env.json` 中 `host` 是否配置。
2. 8080 端口占用：关闭占用进程或改 `server.port`。
3. 启动不建表：检查数据库连接和 `spring.sql.init.*`。
4. 乱码：IDE、终端、数据库统一 UTF-8。
