# Nuist CampusWall 部署说明（开发环境）

## 1. 环境要求
1. JDK 21
2. Maven 3.9+
3. MySQL 8
4. IntelliJ IDEA

## 2. 配置
编辑 `src/main/resources/application.properties`：
1. `spring.datasource.url`
2. `spring.datasource.username`
3. `spring.datasource.password`
4. `jwt.secret` / `jwt.expire-seconds` / `jwt.issuer`

## 3. SQL 初始化
启动时自动执行：
1. `sql/db_init.sql`
2. `sql/db_seed.sql`

## 4. 启动
```bash
mvn spring-boot:run
```

## 5. 接口验收顺序
1. `src/test/http/account.http`
2. `src/test/http/post.http`
3. `src/test/http/comment.http`
4. `src/test/http/like.http`

## 6. 常见问题
1. `http:///api/...`：检查 `http-client.env.json` 里的 `host` 是否为空。
2. 8080 被占用：修改 `server.port` 或关闭占用进程。
3. 启动没建表：检查数据库连接和 `spring.sql.init.*`。
4. 乱码：IDE 和终端统一 UTF-8（Windows 先 `chcp 65001`）。
